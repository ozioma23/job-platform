# Job Platform - Interactive Job Board
**Project Nexus - Frontend (ProDev FE)**

## Overview
This project is an **Interactive Job Board Platform** where users can explore, filter, and apply for job postings.  
The project focuses on:
- **API Integration** with a dummy backend (MockAPI)
- **Advanced Filtering** by category, location, and experience level
- **Responsive and Accessible Design**
- **State Management** with Context API
- **Frontend Development** using Next.js + TypeScript + Tailwind CSS

## Features
1. **Job Listings**
   - Fetch jobs dynamically from API
   - Display job cards with title, company, location, and type
   - Loading and error states included
2. **Job Details Page**
   - Full job description
   - Apply button
3. **Filtering**
   - Filter by category, location, and experience level
4. **Applications**
   - Users can apply for jobs
   - Submitted applications stored (dummy API)
5. **Saved Jobs**
   - Users can bookmark jobs to view later
6. **Profile Page**
   - User information and skills

## Technologies Used
- **Next.js** (React Framework)
- **TypeScript** (Strongly typed code)
- **Tailwind CSS** (Responsive design)
- **Context API** (Global state management)
- **MockAPI** (Interactive dummy backend)
